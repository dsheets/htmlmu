# htmlmu

## Build

Requires [xmlm](http://erratique.ch/software/xmlm) [xmlmu](https://github.com/dsheets/xmlmu) [cmdliner](http://erratique.ch/software/cmdliner).

With OPAM:

```sh
opam install xmlm xmlmu cmdliner
```
